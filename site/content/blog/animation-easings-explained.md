Linear motion looks robotic because nothing in the physical world moves that way. The moment you swap to a soft `power2.out`, an interface stops feeling like software and starts feeling like an object.

My personal hierarchy: `power2.out` for entrances, `power3.in` for exits, custom cubic-bezier when the brand asks for personality. Anything else is a tweak, not a decision.
