A type scale is a contract with your future content. If the scale only works for the demo article, it is not a scale, it is a layout.

I work in three scales: a body scale (1.125 ratio, generous), a UI scale (1.2 ratio, tight), and a display scale (1.5 ratio, brand). Mixing them is fine. Skipping rungs is not.
